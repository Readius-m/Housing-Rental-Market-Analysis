# Housing-Rental-Market-Analysis
项目概述：
使用SQL，Pandas进行数据清洗，多维度分析；构建房屋性价比评分模型。Power bi搭建可视化交互报表。

文件结构：
data/raw - 原始文件
data/output - pandas代码运行后输出结果
script - pandas代码
powerbi - powerbi可视化交互报表

使用步骤：
1.运行 Data/Processing.ipynb 完成数据导入，清洗；创建sqlite库为使用sql进行多维度分析准备。
2.运行 Data/Metric Calculation.ipynb 使用sql对数据进行多维度分析。
3.运行 Data/Listing Rating.ipynb 搭建数据进行性价比评分模型。
4.运行 powerbi/房源分析可视化.pbix 为数据可视化交互报表。
